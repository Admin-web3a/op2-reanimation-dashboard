# ОП Dashboard — Реанимация УП-2 ручная

Автоматический дашборд по сделкам amoCRM для отдела продаж Web3 Academy.

- **Источник:** Рабочий источник = "Реанимация УП-2 ручная" (enum_id 954555)
- **CRM:** simmihur.amocrm.ru
- **Обновление:** ежедневно в 07:00 и 09:00 МСК

## Как работает

1. GitHub Actions запускает `generate_amo_report.py` по расписанию
2. Скрипт тянет сделки из amoCRM через API
3. Генерирует `docs/index.html` с Chart.js дашбордом
4. HTML коммитится и деплоится через GitHub Pages

## Настройка

В репозитории должен быть секрет `AMO_TOKEN` — Long-lived токен от amoCRM.
